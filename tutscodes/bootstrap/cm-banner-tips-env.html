Why use environment file in React JS?

  • 1. To manage the different credentials and information like API URL, Access Codes etc.
  • 2. It makes the development, testing and production deployment process easy.
  • 3. It reduce the changes of accidentally uploading of development / testing credentails to production.
  • 4. We can share the codes with other without sharing the credentails.
